FtdiSharp

A .NET package for interfacing FTDI USB controller ICs

C#/.NET NuGet
FTDI provides a DLL that can be used to read and write pin voltages on FT232 (USB-to-serial controller) ICs, but it is provided as a closed-source DLL with a C++ header file. FtdiSharp is a C# wrapper for providing an idiomatic C# interface to this series of chips, and also provides high-level management of IO protocols like SPI and I2C.
  • FtdiSharp on NuGet
  • FtdiSharp on GitHub

AbfSharp

A .NET package for interfacing Axon Binary Format (ABF) files

C#/.NET NuGet
AbfSharp is a .NET library for reading electrophysiology data from Axon Binary Format (ABF) files. ABFSharp wraps a closed-source DLL and exposes its core functionality with an idiomatic .NET interface.
  • AbfSharp on GitHub
  • AbfSharp on NuGet
  • Enables .NET apps like AbfView and AbfAuto

SciTIF

A .NET package for working with high-dimensional imaging data

C#/.NET NuGet
SciTIF offers an idiomatic .NET API for working with 5-dimensional imaging data stored in TIF files. Unlike most common graphics libraries, it has first-class support for performing lossless mathematics operations on high-depth 16-bit, 24-bit, and 32-bit image data.
  • SciTIF on GitHub
  • SciTIF on NuGet

Websites and Web Applications

ScottPlot.NET

Information and Documentation Website

C#/.NET Azure Functions JavaScript Hugo
The ScottPlot website contains a mix of dynamic and static content which is generated automatically by the CI/CD system (GitHub Actions) when commits are merged to the main branch or new releases are published.
  • Documentation as Code: The ScottPlot Cookbook contains documentation generated automatically from a unit test project which generates images then builds the cookbook pages using source code obtained by parsing the source code files of the unit tests.
  • Changelog Formatting: The ScottPlot Changelog contains rich HTML (links to GitHub issues and users) and images (GitHub user profile pictures) updated from GitHub Actions. A change to changelog.md triggers a workflow that uses a .NET Console application to parse the changelog, generate HTML, and retrieve and resize user profile pictures, then commit changes to the ScottPlot.NET repository.
  • Live Metrics: Despite being a static website, the bottom of ScottPlot.net shows graphs of the current number of package downloads and github stars. Image URLs point to a cloud native application that runs hourly and automatically updates the images when metrics change.
  • Client-Side Interactive Search: The ScottPlot Cookbook Search Tool helps search all cookbook recipes and is implemented using Vanilla JavaScript. The test system that generates the cookbook creates a JSON file containing all recipe titles, descriptions, and permalinks which is consumed by the search tool when it is activated.
  • Build and Deployment: Changes to content of the ScottPlot.NET repository trigger a workflow that converts documentation pages from Markdown to HTML and applies theming and navigation styling to every page of the website using the Hugo static site generator. Deployment is achieved by a GitHub action which synchronizes the build output with the website folder

QRSS Plus

Automatically Updating Active QRSS Grabbers List

Azure Functions React Python PHP AWS SQL Docker
Cloud native web application that displays real-time radio frequency spectrograms from receivers maintained by amateur radio operators around the world. An Azure function runs every 10 minutes and uses MD5 hashes to identify new image data to update cloud storage and generate 8-hour images are stitched together from 10-minute spectrograms.
This application is presently being migrated from a cloud native serverless design running on Azure (Azure Functions and Blob Storage) to a microservices design (ASP.NET Core, NGINX, and MySQL) using Docker in an Elastic Compute Cloud (EC2) instance on AWS.
  • Live web app: https://swharden.com/qrss/plus/
  • Continuously running for over 10 years
  • 100-200 daily users

LJPcalc

Liquid Junction Potential Calculator

C#/.NET Blazor WebAssembly
A Blazor WebAssembly application for calculating the voltage across the liquid junction formed by two solutions with arbitrary electrolyte concentrations. Solutions are achieved by solving the Nernst-Planck set of equations for the stationary state in conditions of local electrical quasi-neutrality at all points of the junction.
  • Live web app: https://swharden.com/LJPcalc/
  • Cited by numerous scientific publications

QRSS Transmitter

Ultrastable Frequency Synthesizer

AVR C/C++ C#/.NET KiCad
Custom design for a ovenized crystal oscillator with varactor-controlled frequency-shift-keying using a custom PCB and SMT components hand-assembled using a microscope. Early tests demonstrated <0.1 PPM stability near 10 MHz using 5 Hz FSK driven using an ATMEL AVR microcontroller.

Talking Frequency Counter

Audible frequency readout for radio operators with impaired vision

AVR C/C++ Python KiCad
A low-cost standalone frequency counter that senses the frequency of the near radio transmission and reads its frequency audibly using a human voice to help radio operators know what frequency their radio is tuned to if visual impairments prevent them from reading the display. This project is a work in progress. An early prototype is pictured, but work is ongoing as I incorporate feedback from a blind radio operator I am collaborating with.
