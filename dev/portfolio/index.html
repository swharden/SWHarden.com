Open Source Packages

ScottPlot

A .NET package for interactively displaying large datasets

ScottPlot can create static plots as images in console applications and server environments, and has 7 additional NuGet packages to provide mouse-interactive plots on a variety of platforms including Windows Forms, WPF, Avalonia, Maui, Blazor, and more
  • The ScottPlot Changelog documents >1.5k improvements made across nearly 200 releases since 2019
  • ScottPlot Packages on NuGet - over 1.8 million installs
  • ScottPlot on GitHub - used in 1.9k projects, resolved 4k issues and pull requests
  • Visit ScottPlot.NET for more information
  • Technology: C#/.NET, NuGet

pyABF

A python package for reading data from Axon Binary Format (ABF) files

I reverse-engineered a proprietary file format used for recording electrical data in neurophysiology experiments and created a Pythonic API to expose information and signal data in these files. This project enables automated data analysis for projects that previously required manual analysis using a proprietary windows-only application.
  • Continuously maintained since 2017
  • Cited in numerous scientific publications
  • pyABF website
  • pyABF Package (PyPi)
  • pyABF on GitHub
  • Technology: Python, PyPi

FFTsharp

A .NET package for computing the Fast Fourier Transform (FFT) of real or complex data

FftSharp provides methods for calculating the FFT using a MIT-licensed .NET Standard library which can be easily used in proprietary and cross-platform applications. It also contains methods to facilitate data windowing and frequency spectrum filtering.
  • FftSharp on NuGet (>130k installs)
  • FftSharp on GitHub (used in >230 projects)
  • Technology: C#/.NET, NuGet

SwarmFit

SwarmFit is a .NET package for fitting arbitrary curve formulas to X/Y data points using particle swarm optimization. Unlike other gradient decent strategies, finding the derivative of the error function is not required.
  • Theory (Article)
  • SwarmFit on GitHub
  • SwarmFit on NuGet
  • Technology: C#/.NET, NuGet

FtdiSharp

A .NET package for interfacing FTDI USB controller ICs

FTDI provides a DLL that can be used to read and write pin voltages on FT232 (USB-to-serial controller) ICs, but it is provided as a closed-source DLL with a C++ header file. FtdiSharp is a C# wrapper for providing an idiomatic C# interface to this series of chips, and also provides high-level management of IO protocols like SPI and I2C.
  • FtdiSharp on NuGet
  • FtdiSharp on GitHub
  • Technology: C#/.NET, NuGet

AbfSharp

A .NET package for interfacing Axon Binary Format (ABF) files

AbfSharp is a .NET library for reading electrophysiology data from Axon Binary Format (ABF) files. ABFSharp wraps a closed-source DLL and exposes its core functionality with an idiomatic .NET interface.
  • AbfSharp on GitHub
  • AbfSharp on NuGet
  • Enables .NET apps like AbfView and AbfAuto
  • Technology: C#/.NET, NuGet

SciTIF

A .NET package for working with high-dimensional imaging data

SciTIF offers an idiomatic .NET API for working with 5-dimensional imaging data stored in TIF files. Unlike most common graphics libraries, it has first-class support for performing lossless mathematics operations on high-depth 16-bit, 24-bit, and 32-bit image data.
  • SciTIF on GitHub
  • SciTIF on NuGet
  • Technology: C#/.NET, NuGet

NumberSpeaker

Arduino library for reading numbers with a speaker

Allows numbers stored in memory to be read out loud using a speaker connected to a PWM output pin. Recordings were carefully sampled and compressed to fit in extremely limited program memory.
  • Theory and implementation (Article)
  • NumberSpeaker on GitHub
  • NumberSpeaker on Arduino website
  • Technology: C/C++, Python

Websites and Web Applications

ScottPlot.NET

Information and Documentation Website

The ScottPlot website contains a mix of dynamic and static content which is generated automatically by the CI/CD system (GitHub Actions) when commits are merged to the main branch or new releases are published.
  • Documentation as Code: The ScottPlot Cookbook contains documentation generated automatically from a unit test project which generates images then builds the cookbook pages using source code obtained by parsing the source code files of the unit tests.
  • Changelog Formatting: The ScottPlot Changelog contains rich HTML (links to GitHub issues and users) and images (GitHub user profile pictures) updated from GitHub Actions. A change to changelog.md triggers a workflow that uses a .NET Console application to parse the changelog, generate HTML, and retrieve and resize user profile pictures, then commit changes to the ScottPlot.NET repository.
  • Live Metrics: Despite being a static website, the bottom of ScottPlot.net shows graphs of the current number of package downloads and github stars. Image URLs point to a cloud native application that runs hourly and automatically updates the images when metrics change.
  • Client-Side Interactive Search: The ScottPlot Cookbook Search Tool helps search all cookbook recipes and is implemented using Vanilla JavaScript. The test system that generates the cookbook creates a JSON file containing all recipe titles, descriptions, and permalinks which is consumed by the search tool when it is activated.
  • Build and Deployment: Changes to content of the ScottPlot.NET repository trigger a workflow that converts documentation pages from Markdown to HTML and applies theming and navigation styling to every page of the website using the Hugo static site generator. Deployment is achieved by a GitHub action which synchronizes the build output with the website folder
  • Technology: C#/.NET, Azure Functions, HTML, CSS, JavaScript, Hugo

QRSS Plus

Automatically Updating Active QRSS Grabbers List

Cloud native web application that displays real-time radio frequency spectrograms from receivers maintained by amateur radio operators around the world. An Azure function runs every 10 minutes and uses MD5 hashes to identify new image data to update cloud storage and generate 8-hour images are stitched together from 10-minute spectrograms.
  • Live web app: https://swharden.com/qrss/plus/
  • Continuously running for over 10 years
  • 100-200 daily users
  • Technology: Azure Functions, React
  • Previously: Python, PHP

LJPcalc

Liquid Junction Potential Calculator

A Blazor WebAssembly application for calculating the voltage across the liquid junction formed by two solutions with arbitrary electrolyte concentrations. Solutions are achieved by solving the Nernst-Planck set of equations for the stationary state in conditions of local electrical quasi-neutrality at all points of the junction.
  • Live web app: https://swharden.com/LJPcalc/
  • Cited by numerous scientific publications
  • Technology: C#/.NET, Blazor WebAssembly

A locally-run web application that pairs automatic data analysis with project organization and experiment annotation by teams of scientists. Manages over 200k electrical recordings acquired over more than 20 years to present results interactively in an application used by more than 50 researchers.
  • Technology: ASP.NET core, JavaScript, SQL
  • Previously: Blazor Server, SQL
  • Previously: Python, PHP, SQL

Real-Time Radio Frequency Spectrogram

FSKview is a high-resolution spectrogram for viewing narrow-band frequency-shift keyed radio signals in real time. It can be paired with WSJT-X software to visualize encoded data. It can automatically upload QRSS images for display on the QRSS Plus website.

Dendrite Tracer

Multidimensional Fluorescence Analysis

A tool for measuring fluorescence intensity along structures in 3D space from 5-dimensional microscopy data containing multi-channel time-series images. Enables rapid hypothesis testing and report generation to minimize the time required to draw conclusions after data acquisition.

Tmoji

Emoji and symbols in the system tray

Tmoji is a Windows tray icon that makes it easy to copy emoji and special characters to the clipboard. Unlike the Win + . shortcut, TMoji allows custom positioning for commonly used emoji, symbols, and special characters for rapid recall.

Ratio5D

5-dimensional video analysis

Ratio5D is a desktop application for rapidly drawing conclusions from live-cell imaging involving 5-dimensional video data (3D space, time, and wavelength). Enables rapid assessment of experimental outcome to minimize the time required to draw conclusions after data acquisition.

BasicTimer

Minimal timer for Windows

BasicTimer allows rapid startup, placement, and control of simple always-on-top timers with a variety of customization options. Color settings, alarms, and progressbar rate can be styled to facilitate monitoring multiple timers running concurrently.

Scan-A-Gator

Live Neuron Fluorescence Analysis

A Windows desktop application for analyzing multi-channel time-series fluorescent imaging data from live neurons. Collapses 5-dimensional imaging data into a simple interactive interface that enables rapid assessment, report generation, and data transformation.
  • Enabled experiments underlying Sheng et al., 2021
  • Technology: C#/.NET, Python

Electrophysiology Auto-Analysis

Automated analysis pipeline for cellular electrical signals

AbfAuto is a suite of tools that runs on a server to provide real-time data analysis and report generation capabilities. A filesystem monitor identifies newly created data files, they are analyzed automatically according to experiment designs defined in code, results are stored in image files, report files, SQL database records, and are exposed to the user via a web interface using an ASP.NET core backend and Vanilla JavaScript frontend.
  • AbfAuto on GitHub (C#)
  • AbfAuto on GitHub (Python)
  • Technology: C#/.NET, Python, SQL

IoT Attic Temperature Sensor

Web-Enabled Device for Remotely Monitoring Indoor/Attic Temperatures

A project that pairs BMP280 temperature sensors with ESP8266 WiFi-enabled microcontrollers to continuously report indoor, outdoor, and attic temperature to a web-accessible endpoint allowing worldwide monitoring of real-time home temperature information. Moisture encroachment following a 2024 hurricane disabled the outdoor sensor, but the indoor and attic sensors remain steadfast.
  • Live Dashboard (internal tool)
  • GitHub Project
  • Technology: C/C++, SQL, PHP, JavaScript

Breathing Monitor

Respiration monitor for infants and small animals

USB interface provides power and PC connectivity for streaming data and customizing filter settings. Built-in breathing rate detection allows breath rate monitoring and analog signal filtering and amplification without requiring a computer.
  • Design notes and software (GitHub)
  • Technology: C/C++, AVR DD-Series MCU, C#/.NET

QRSS Transmitter

Ultrastable Frequency Synthesizer

Custom design for a ovenized crystal oscillator with varactor-controlled frequency-shift-keying using a custom PCB and SMT components hand-assembled using a microscope. Early tests demonstrated <0.1 PPM stability near 10 MHz using 5 Hz FSK driven using an ATMEL AVR microcontroller.
  • Oscillator design notes (Article)
  • Keyer design and software (GitHub)
  • QRSS Hardware on GitHub
  • Technology: C/C++, C#/.NET, KiCad

Sound Card ECG

Low Cost Electrical Heart Monitor

Multiple designs for low-cost ECG circuits that amplify electrical signals from the heart, digitize them using the sound card, and perform real-time signal filtering and analysis using custom software.
  • AD8232 design Article and YouTube Video
  • discrete op-amp design Article and YouTube Video
  • Technology: C#/.NET, Python

Talking Frequency Counter

Audible frequency readout for radio operators with impaired vision

A low-cost standalone frequency counter that senses the frequency of the near radio transmission and reads its frequency audibly using a human voice to help radio operators know what frequency their radio is tuned to if visual impairments prevent them from reading the display. This project is a work in progress. An early prototype is pictured, but work is ongoing as I incorporate feedback from a blind radio operator I am collaborating with.
  • Project Files on GitHub
  • Technology: C/C++, AVR DD-Series MCU
