ScottPlot

A .NET package for interactively displaying large datasets

ScottPlot can create static plots as images in console applications and server environments, and has 7 additional NuGet packages to provide mouse-interactive plots on a variety of platforms including Windows Forms, WPF, Avalonia, Maui, Blazor, and more
  • The ScottPlot Changelog documents >1.5k improvements made across nearly 200 releases since 2019
  • ScottPlot Packages on NuGet - over 1.8 million installs
  • ScottPlot on GitHub - used in 1.9k projects, resolved 4k issues and pull requests
  • Visit ScottPlot.NET for more information
  • Technology: C#/.NET, NuGet

ScottPlot.NET

Information and Documentation Website

The ScottPlot website contains a mix of dynamic and static content which is generated automatically by the CI/CD system (GitHub Actions) when commits are merged to the main branch or new releases are published.
  • Documentation as Code: The ScottPlot Cookbook contains documentation generated automatically from a unit test project which generates images then builds the cookbook pages using source code obtained by parsing the source code files of the unit tests.
  • Changelog Formatting: The ScottPlot Changelog contains rich HTML (links to GitHub issues and users) and images (GitHub user profile pictures) updated from GitHub Actions. A change to changelog.md triggers a workflow that uses a .NET Console application to parse the changelog, generate HTML, and retrieve and resize user profile pictures, then commit changes to the ScottPlot.NET repository.
  • Build and Deployment: Changes to content of the ScottPlot.NET repository trigger a workflow that converts documentation pages from Markdown to HTML and applies theming and navigation styling to every page of the website using the Hugo static site generator. Deployment is achieved by a GitHub action which synchronizes the build output with the website folder
  • Client-Side Interactive Search: The ScottPlot Cookbook Search Tool helps search all cookbook recipes and is implemented using Vanilla JavaScript. The test system that generates the cookbook creates a JSON file containing all recipe titles, descriptions, and permalinks which is consumed by the search tool when it is activated.
  • Technology: C#/.NET, HTML, CSS, JavaScript, Hugo

pyABF

A python package for reading data from Axon Binary Format (ABF) files

I reverse-engineered a proprietary file format used for recording electrical data in neurophysiology experiments and created a Pythonic API to expose information and signal data in these files. This project enables automated data analysis for projects that previously required manual analysis using a proprietary windows-only application.
  • Continuously maintained since 2017
  • Cited in numerous scientific publications
  • pyABF website
  • pyABF Package (PyPi)
  • pyABF on GitHub
  • Technology: Python, PyPi

FFTsharp

A .NET package for computing the Fast Fourier Transform (FFT) of real or complex data

FftSharp provides methods for calculating the FFT using a MIT-licensed .NET Standard library which can be easily used in proprietary and cross-platform applications. It also contains methods to facilitate data windowing and frequency spectrum filtering.
  • FftSharp on NuGet (>130k installs)
  • FftSharp on GitHub (used in >230 projects)
  • Technology: C#/.NET, NuGet

FtdiSharp

A .NET package for interfacing FTDI USB controller ICs

FTDI provides a DLL that can be used to read and write pin voltages on FT232 (USB-to-serial controller) ICs, but it is provided as a closed-source DLL with a C++ header file. FtdiSharp is a C# wrapper for providing an idiomatic C# interface to this series of chips, and also provides high-level management of IO protocols like SPI and I2C.
  • FtdiSharp on NuGet
  • FtdiSharp on GitHub
  • Technology: C#/.NET, NuGet

QRSS Plus

Automatically Updating Active QRSS Grabbers List

Cloud native web application that displays real-time radio frequency spectrograms from receivers maintained by amateur radio operators around the world. An Azure function runs every 10 minutes and uses MD5 hashes to identify new image data to update cloud storage and generate 8-hour images are stitched together from 10-minute spectrograms.
  • Live web app: https://swharden.com/qrss/plus/
  • Continuously running for over 10 years
  • 100-200 daily users
  • Technology (2018-present): Azure Functions, React
  • Technology (2013-2018): Python, PHP
